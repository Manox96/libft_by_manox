# Part 1 - Libc Functions

## ft_isalpha
Description: Checks if a character is an alphabetic character (A-Z, a-z).
## ft_isdigit
Description: Checks if a character is a digit (0-9).
## ft_isalnum
Description: Checks if a character is alphanumeric (alphabetic or numeric).
## ft_isascii
Description: Checks if a character is a valid ASCII character (0-127).
## ft_isprint
Description: Checks if a character is a printable character (including space).
## ft_strlen
Description: Calculates the length of a string.
## ft_memset
Description: Fills a block of memory with a specified value.
## ft_bzero
Description: Sets the first n bytes of a block of memory to zero.
## ft_memcpy
Description: Copies n bytes from memory area src to memory area dest.
## ft_memmove
Description: Copies n bytes from memory area src to memory area dest, handling overlapping.
## ft_strlcpy
Description: Copies a string to a specified size, ensuring null-termination.
## ft_strlcat
Description: Concatenates strings, ensuring that the total length does not exceed a specified size.
## ft_toupper
Description: Converts a lowercase letter to uppercase.
## ft_tolower
Description: Converts an uppercase letter to lowercase.
## ft_strchr
Description: Locates the first occurrence of a character in a string.
## ft_strrchr
Description: Locates the last occurrence of a character in a string.
## ft_strncmp
Description: Compares the first n characters of two strings.
## ft_memchr
Description: Locates the first occurrence of a character in a block of memory.
## ft_memcmp
Description: Compares two blocks of memory.
## ft_strnstr
Description: Locates a substring in a string, searching a limited number of characters.
## ft_atoi
Description: Converts a string to an integer.
### Part 2 - Additional Functions
## ft_substr
Description: Allocates and returns a substring from a string, starting at a specified index and of a specified length.
## ft_strjoin
Description: Allocates and returns a new string resulting from the concatenation of two input strings.
## ft_strtrim
Description: Allocates and returns a copy of a string with specified characters removed from the beginning and end.
## ft_split
Description: Allocates and returns an array of strings obtained by splitting a string using a specified delimiter.
## ft_itoa
Description: Allocates and returns a string representing an integer.
## t_strmapi
Description: Applies a function to each character of a string, creating a new string resulting from successive applications of the function.
## ft_striteri
Description: Applies a function to each character of a string, passing its index as the first argument.
## ft_putchar_fd
Description: Outputs a character to a specified file descriptor.
## ft_putstr_fd
Description: Outputs a string to a specified file descriptor.
## ft_putendl_fd
Description: Outputs a string followed by a newline to a specified file descriptor.
## ft_putnbr_fd
Description: Outputs an integer to a specified file descriptor.
### Bonus Part
## ft_lstnew
Description: Allocates and returns a new node for a linked list.
## ft_lstadd_front
Description: Adds a node to the front of a linked list.
## ft_lstsize
Description: Counts the number of nodes in a linked list.
ft_lstlast
Description: Returns the last node of a linked list.
## ft_lstadd_back
Description: Adds a node to the end of a linked list.
## ft_lstdelone
Description: Deletes a node from a linked list, freeing the memory of its content.
## ft_lstclear
Description: Deletes and frees all nodes of a linked list.
## ft_lstiter
Description: Iterates a linked list and applies a function to the content of each node.
## ft_lstmap
Description: Iterates a linked list, applies a function to the content, and creates a new list from the results.
